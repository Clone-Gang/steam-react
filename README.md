# Steam React

---

簡易版 Steam 前端克隆使用 React、Redux 和 TailwindCSS。

> 不屬於 Steam® 版權所有，為 Dalufishe 開發，並不以營利為目的之開源項目。

### 使用應用程序

---

線上版網址: <a href=""></a>

`注意:` 須先下載 NodeJS 運行環境 (最新版本) 及 npm 套件管理工具。

- 克隆此專案

```bash
git clone https://github.com/Dalufishe/steam-react
```

- 下載依賴

```bash
npm install
```

- 啟動項目

```bash
npm run start
```

happy hacking !
