# Steam React

---

簡易版 Steam 前端克隆使用 React、Redux 和 TailwindCSS。

---

目前版本 (開發中版本) 為單純頁面克隆，使用假數據，還不具有實際功能。

### 運行應用程序

---

> _注意，此處為開發者文檔_

您須先建置 NodeJS 運行環境 (最新版本) 及 npm 套件管理工具才能運行正常此應用程序。

- 克隆此專案

```bash
git clone https://github.com/Clone-Gang/steam-react
```

- 下載相關依賴

```bash
npm install
```

- 啟動項目

```bash
npm run start
```

您應該就可以正常正常啟動項目。(本機位置，3000 端口)
